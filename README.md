# CNN

implement CNN by using keras and tensorflow

## how to use

1. write config file as yaml format
2. prepare images like below

> ```
> |---images/
>        |---dog/
>             |---1.png
>             |---2.png
>             |---...
>        |---cat/
>             |---1.png
>             |---2.png
>             |---...
>        |---horse/
>             |---1.png
>             |---2.png
>             |---...
> ```


3. execute runner (if you want to see help, use -h option)
